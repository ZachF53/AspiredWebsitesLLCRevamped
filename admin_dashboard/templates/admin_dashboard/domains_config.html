How the toggle works

  1. The flag lives on the NamecheapConfig singleton row in the DB — it's NOT an env var.
  2. Every API call calls NamecheapConfig.is_sandbox() at request time. No restart needed after toggling.
  3. The get_client() helper does NOT cache — every call gets a fresh client targeting the current env.
  4. On first deploy, the singleton seeds from settings.NAMECHEAP_SANDBOX in .env; thereafter the DB wins.
  5. Every toggle is logged at WARNING level + recorded above with timestamp + admin username.
