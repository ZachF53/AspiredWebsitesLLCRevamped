Step 2b — Generate SSH Vault Key (manual, non-Stripe Droplets only)

Stripe-triggered Droplets get this automatically. Run this only when you provisioned the box yourself (no Stripe webhook fired). Generates an Ed25519 keypair on the server, authorises it for root, captures the private key, and locks down password auth.

Paste the printed private key into Vault → {{DOMAIN}} → Add Credential, category Server / Hosting, auth type Private Key. Do not save the private key anywhere else.

Step 3 — Install Dependencies
Step 3b — Install PDF Generation Libraries

Required for WeasyPrint PDF generation. Without these libs, scan reports and monthly reports silently fall back to HTML attachments — emails go out with .html files instead of .pdf. The base snapshot does NOT include these, so every new server needs this step.

Verify with: /var/www/aspired/venv/bin/python -c "import weasyprint; print(weasyprint.__version__)" — if it imports without an OSError about libpango/libcairo, the libs are in place.
